# Videos de JYP Turismo

Coloca aquí los videos promocionales de tu empresa.

## Videos sugeridos:
- `hero-video.mp4` - Video principal para el header (15-30 segundos)
- `presentacion-empresa.mp4` - Video de presentación de la empresa
- `tour-virtual.mp4` - Tour virtual de destinos

## Formatos recomendados:
- Formato: MP4 (H.264)
- Resolución: 1920x1080px (Full HD)
- Duración: 15-60 segundos
- Tamaño: Máximo 10MB por video
- FPS: 30fps

## Optimización:
Para optimizar el tamaño del video, puedes usar herramientas como:
- HandBrake (gratuito)
- Adobe Media Encoder
- FFmpeg (comando): `ffmpeg -i input.mp4 -vcodec h264 -acodec mp3 output.mp4`
